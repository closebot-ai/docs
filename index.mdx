## Authentication

All API requests require authentication using a Key token. Include your API token in the `Authorization` header of every request.

```bash
X-CB-KEY: YOUR_API_KEY
```

<Warning>
Keep your API token secure and never expose it in client-side code or public repositories.
</Warning>

## Base URL

All API requests should be made to:

```
https://api.closebot.com
```
